# Analyze Guidelines Skill

지원사업 모집요강 분석 및 전략적 사업계획서 최적화

## Purpose

지원사업 모집요강(PDF/MD)을 분석하여 평가자의 의도를 파악하고, 기존 사업계획서를 평가 기준에 맞게 전략적으로 재구성합니다.

**핵심 철학**: 좋은 아이디어를 먼저 깊이 개발하고(Phase 0-5), 마지막에 요구사항에 맞게 설득력 있게 포장(Phase 6)

## When to Use

- 지원사업 공고문/모집요강 분석
- 평가 기준 및 배점 추출
- 가점 항목 체크리스트 생성
- 필수 섹션 및 형식 요건 파악
- 사업계획서 → 모집요강 매핑
- 설득력 있는 최종 문서 작성

## Usage

### 기본 사용법

```bash
# PDF 파일 분석
/analyze-guidelines "path/to/모집요강.pdf"

# Markdown 파일 분석
/analyze-guidelines "path/to/공고문.md"

# 특정 산업/분야 컨텍스트 추가
/analyze-guidelines "모집요강.pdf" --industry "화장품"
```


### 분석 프롬프트

```markdown
당신은 정부 지원사업 전문 컨설턴트입니다.

주어진 모집요강을 분석하여:
1. 평가자의 숨은 의도를 파악하세요
2. 선정 가능성을 높이는 전략을 도출하세요
```

## Error Handling

| 에러 | 원인 | 해결책 |
|-----|------|--------|
| PDF_READ_ERROR | PDF 파싱 실패 | 텍스트 추출 도구 사용 |
| GUIDELINES_NOT_PROVIDED | 모집요강 미제공 | 일반 양식으로 진행 |
| FORMAT_UNKNOWN | 양식 인식 불가 | 수동 분석 가이드 제공 |

## Fallback Strategy

모집요강이 제공되지 않은 경우:
1. 일반적인 지원사업 양식 적용
2. 범용 평가 기준 기반 최적화
3. 사용자에게 양식 확인 요청
